# Asterisk Verification Dashboard

Панель управления для системы верификации клиентов Asterisk с интеграцией Telegram.

## 🚀 Возможности

- 📊 Дашборд со статистикой звонков
- 📞 Просмотр логов верификации
- 👥 Управление клиентами (CRUD)
- 📈 Графики и аналитика
- 📤 Импорт клиентов из CSV
- 🤖 Интеграция с Telegram

## 🛠 Технологии

- Django 4.2
- AdminLTE 4
- PostgreSQL
- Docker
